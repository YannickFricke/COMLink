# COMLink

A simple communication system for "Pen and Paper" games like Shadowrun.

This is a simple chat application where players can manage and message their contacts.

CAUTION: The only supported language is german.

## Features

- Players
    - Manage their own contacts
    - Message the contacts
    - Edit own messages
    - Remove own messages
- Gamemaster
    - List all contacts for each player
    - Edit the contact names
    - Writing messages on behalf of contacts
    - Edit messages (from players and contacts)
    - Remove messages (from players and contacts)

## Technology stack

- TypeScript
- Express
- React.JS
- Socket.IO
- Redux
- SQLite
- Knex.js
